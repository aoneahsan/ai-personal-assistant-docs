# Use the Pomodoro timer

**This guide shows how to focus with AIPA's Pomodoro Timer — run a 25-minute work cycle, take a short break, repeat, and review your focus stats.** The Pomodoro technique trades long, leaky work sessions for short, protected sprints.

## Steps

1. **Open the Pomodoro Timer.** From the Features launcher, search "pomodoro" and open it.
2. **Pick one task.** Choose a single task from your [Daily Planner](/guides/plan-your-day) — one sprint, one task.
3. **Start a 25-minute work cycle.** Begin the timer and work only on that task until it rings.
4. **Take a short break.** Step away for the break cycle — water, stretch, look away from the screen.
5. **Repeat.** After a few cycles, take a longer break.
6. **Check your stats.** Review how many focus cycles you completed to see your real focused time.

## Tips

- **Close distractions before you start.** A clean start makes the 25 minutes count.
- **One task per sprint.** If a new task pops up, jot it in [Quick Notes](/features/productivity) and return to your sprint.
- **Honor the break.** Breaks are part of the method, not wasted time — they're what make the next sprint sharp.
- **Pair with the [Time Tracker](/features/productivity)** if you want a fuller picture of where time goes.

## FAQ

### Can I change the cycle lengths?
The timer runs work/break cycles; use the timer's options to fit the rhythm that works for you.

### Does it track my focus over time?
Yes — it keeps focus stats so you can see completed cycles.

### Will it keep running if I switch tabs?
Keep the app open for the timer to run reliably; behavior can vary by browser/device when backgrounded.
