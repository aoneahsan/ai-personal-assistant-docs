# Productivity Tools

**The Productivity category is AIPA's set of organize-your-day tools — task lists, notes, planners, goals, projects, reminders, a calendar, bookmarks, a mind map, a focus timer, and a habit tracker — that cover the everyday work of planning and getting things done.** Each tool stores its data under your account so it syncs across devices.

## The tools

| Tool | What it does |
| --- | --- |
| **Tasks** | A smart to-do list with priorities, categories, and reminders. |
| **Quick Notes** | Fast note-taking with search and organization. |
| **Rich Text Notes** | A note editor with formatting and attachments. |
| **Diary & Journal** | A personal diary with mood tracking. |
| **Goals** | Set, track, and achieve personal and professional goals. |
| **Projects** | A Kanban-style board to organize work into columns. |
| **Daily Planner** | Plan your day and review what you accomplished. |
| **Weekly / Monthly Planner** | Longer-range planning and scheduling. |
| **Time Tracker** | Track time spent on tasks and projects. |
| **Reminders** | Set reminders so you don't miss events and tasks. |
| **Smart Calendar** | A calendar with event management. |
| **Bookmark Manager** | Save and organize web bookmarks. |
| **Mind Map** | Visual brainstorming and idea organization. |
| **Pomodoro Timer** | A focus timer with work/break cycles and stats. |
| **Habit Tracker** | Build daily habits and track streaks. |

## Common use cases

- **Run your day end to end.** Capture tasks in **Tasks**, lay them out in the **Daily Planner**, focus with the **Pomodoro Timer**, and review at day's end.
- **Keep a second brain.** Use **Quick Notes** for fast capture and **Rich Text Notes** for longer, formatted writing with attachments; reach back via search.
- **Build routines that stick.** Add habits to the **Habit Tracker**, set **Reminders**, and watch your streaks grow.
- **Organize a project.** Break work into a **Projects** Kanban board, track effort with the **Time Tracker**, and keep the deadline in the **Smart Calendar**.
- **Think visually.** Sketch ideas in the **Mind Map**, then turn branches into **Goals** and **Tasks**.

## What these tools do and don't do

- They are **personal-productivity tools**, designed for individual planning. The **Projects** board is Kanban-style organization, not a full team project-management suite with assignees and permissions.
- **Reminders and notifications** rely on your device allowing notifications; the app requests that permission only when you opt in.
- Data persists and syncs when you're **signed in**; see [Account & Sync](/getting-started/account-and-sync).

## FAQ

### Do the productivity tools use AI?
No — these are non-AI tools and work without a Gemini key. AI tools live in the [AI & Content](/features/ai-and-content) category.

### Can I sync tasks and notes across devices?
Yes, when signed in with the same Google account. See [Account & Sync](/getting-started/account-and-sync).

### Is the Projects board for teams?
It's a personal Kanban board for organizing your own work, not a multi-user team tool.

### Do reminders work if notifications are off?
Reminders need notification permission to alert you. You can still see upcoming items inside the app, but push alerts require allowing notifications.
