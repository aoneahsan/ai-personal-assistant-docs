# Utility Tools

**The Utilities category is AIPA's grab-bag of small, frequently used tools — PDF annotation, a cloud file manager, file compression, QR codes, currency and time-zone conversion, an event countdown, and a gratitude journal.** These are the quick utilities you'd otherwise install a separate single-purpose app for.

## The tools

| Tool | What it does |
| --- | --- |
| **PDF Annotator** | Annotate and mark up PDF files. |
| **Cloud File Manager** | Manage your files in the cloud. |
| **File Tools** | Compress and decompress file archives. |
| **QR Code Studio** | Generate and scan QR codes. |
| **Currency Converter** | Convert between currencies. |
| **Event Countdown** | Count down to important events. |
| **Time Zone Converter** | Convert times across time zones. |
| **Gratitude Journal** | Practice daily gratitude with short entries. |

## Common use cases

- **Mark up a document.** Open a PDF in the **PDF Annotator**, add notes and highlights, and save your marked-up copy.
- **Share a link fast.** Generate a **QR code** for a URL, or scan one with your camera.
- **Plan across time zones.** Use the **Time Zone Converter** to find a meeting time that works, and set an **Event Countdown** for the date.
- **Handle money math.** Convert prices with the **Currency Converter** while shopping or budgeting.
- **End the day well.** Write a line in the **Gratitude Journal**.

## Honest notes

- **File uploads use FilesHub.** The cloud file manager stores uploaded files via AIPA's file-storage service (FilesHub), not on your device alone. Deleting files in the app removes them from storage. See [Privacy & Data](/privacy-and-data).
- **The QR scanner needs camera access**, which the app requests only when you open the scanner. You can generate QR codes without the camera.
- **Currency conversion uses exchange-rate data** that may not be real-time to the second; use it as a practical estimate, not for financial settlement.
- **These are utilities, not professional software.** The PDF annotator handles everyday markup, not advanced document workflows.

## FAQ

### Does the QR scanner need permission?
Yes — it needs camera access, requested when you open the scanner. Generating QR codes does not need the camera.

### Where are uploaded files stored?
Through AIPA's file-storage service (FilesHub). See [Privacy & Data](/privacy-and-data) for details and deletion.

### Are exchange rates live?
They're practical estimates from exchange-rate data, suitable for everyday conversion rather than exact financial settlement.

### Do utilities sync across devices?
Tool data and uploads tied to your account sync when you're signed in. See [Account & Sync](/getting-started/account-and-sync).
