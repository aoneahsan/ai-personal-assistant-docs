# Learning & Collaboration Tools

**The Learning & Collaboration category is AIPA's set of study- and knowledge-building tools — flashcards, quizzes, a vocabulary builder, a personal wiki, a code-snippet store, polls, a whiteboard, and a speed-reading trainer.** They help you learn, remember, and organize what you know.

## The tools

| Tool | What it does |
| --- | --- |
| **Flashcards** | Create and study flashcard decks. |
| **Quiz Creator** | Build and take quizzes and tests. |
| **Vocabulary Builder** | Collect and learn new words. |
| **Wiki & Knowledge Base** | Personal knowledge management — your own wiki. |
| **Code Snippets** | Store and organize reusable code snippets. |
| **Polls & Surveys** | Create and share polls. |
| **Whiteboard** | A drawing canvas for sketching and collaboration. |
| **Speed Reading** | Improve reading speed with RSVP-style training. |

## Common use cases

- **Study for an exam.** Build **Flashcards**, drill them in study mode, and test yourself with the **Quiz Creator**.
- **Grow your vocabulary.** Add words to the **Vocabulary Builder** and review them over time.
- **Keep a personal knowledge base.** Document what you learn in the **Wiki**, and stash reusable code in **Code Snippets**.
- **Gather opinions.** Spin up a **Poll** and share it.
- **Sketch and think.** Use the **Whiteboard** to diagram ideas.
- **Read faster.** Train with the **Speed Reading** RSVP tool.

## Honest notes on collaboration

- **The whiteboard's real-time collaboration needs network access and a signaling path.** When that isn't available — for example offline — it degrades to local use: you can still draw, but live multi-user collaboration isn't possible without connectivity.
- **These are personal-scale tools.** Polls and the whiteboard are designed for lightweight sharing, not enterprise classroom or workshop management.
- **Your study data syncs** when signed in; see [Account & Sync](/getting-started/account-and-sync).

## FAQ

### Can the whiteboard be used by multiple people at once?
Real-time collaboration requires network access and a signaling connection. Without it, the whiteboard still works locally for your own drawing, but live co-editing isn't available.

### Do flashcards support spaced study?
You create decks and study them in study mode. The tool is for building and reviewing decks rather than a full spaced-repetition scheduling engine.

### Where are my code snippets stored?
Under your own account, syncing across devices when signed in. See [Account & Sync](/getting-started/account-and-sync).

### Do these tools need a Gemini key?
No — they're non-AI tools and work without a key.
